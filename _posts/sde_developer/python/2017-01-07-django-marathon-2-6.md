How to create Database
?

- -
  Makemigrations : create package and send to
  database
- -
  Sqlmigrate : display sql statement done by
  migrate
- -
  Migrate: take the package and create
  database



Command for Migration

-
python manage.py makemigration  --name migration\_name  app

-
python manage.py showmigrations app  [to show all migrations done yet  ]

-
python manage.py sqlmigrate app  0001\_initial
[to sql command generated ]

-
python manage.py migrate



-
python manage.py app  migrate app 0001\_initial
